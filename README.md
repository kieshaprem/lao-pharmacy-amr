# Lao Pharmacy Antibiotic Stewardship

Analysis of antibiotic stewardship practices and drug quality in private pharmacies in Lao PDR, combining field audits, qualitative interviews, and statistical modelling.

This repository contains the code and analytical workflow for a mixed-methods study assessing antibiotic use, regulatory compliance, and antibiotic quality in private pharmacies in Lao PDR.

The study integrates:
- Good Pharmacy Practice (GPP) inspections
- In-depth interviews with pharmacy staff (qualitative analyses not included in this repository)
- Laboratory analysis of antibiotic samples
- Statistical modelling of pharmacy-level outcomes

## Objectives
- To assess compliance with Good Pharmacy Practice (GPP) standards  
- To evaluate the presence of unregistered antibiotics  
- To examine the distribution of WHO AWaRe “Watch” antibiotics  
- To identify structural factors associated with antibiotic availability patterns  

## Methods

Quantitative analysis was conducted using Poisson regression models with an offset for the total number of antibiotics audited per pharmacy. Models examined associations between pharmacy characteristics (e.g., class, location) and:
- Number of unregistered antibiotics  
- Number of WHO “Watch” antibiotics  

Qualitative data from semi-structured interviews were analysed separately to contextualise pharmacy practices and decision-making.
